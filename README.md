Perpvia Telegram Bot

Points bot for the Perpvia community: daily check-in, active-chat, and invite
rewards feed into an NFT Whitelist shop where users link a wallet and redeem
points for WL spots (admin-reviewed). Personal commands work in DM or in the
group; in-group usage auto-cleans up after a few seconds to keep the chat quiet.

See `bot.py` for the full command list and configuration (env vars: BOT_TOKEN,
ADMIN_IDS, GROUP_USERNAME, GROUP_CHAT_ID, DB_PATH, GROUP_MSG_TTL_SECONDS).
